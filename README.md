# dconsole homebrew tap

Homebrew formulas for [dconsole](https://github.com/dconsole/dconsole) and
related tools. Formulas under `Formula/` are auto-generated and pushed
by [GoReleaser](https://goreleaser.com) on every release — don't edit
them by hand.

## Install

```sh
brew tap dconsole/tap
brew install dconsole
```

`brew tap dconsole/tap` is shorthand for `brew tap dconsole/homebrew-tap`
— Homebrew strips the `homebrew-` prefix when you reference a tap.

## Update

```sh
brew update
brew upgrade dconsole
```
